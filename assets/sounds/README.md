# Audio Files

This directory should contain the following audio files:

- `ambient.mp3` - Background horror music (looping)
- `click.mp3` - UI click sound effect
- `success.mp3` - Success notification sound

## Where to Get Audio Files

You can find free horror-themed audio from:
- [Freesound.org](https://freesound.org/)
- [OpenGameArt.org](https://opengameart.org/)
- [ZapSplat](https://www.zapsplat.com/)

## Note

The application will work without audio files. Sound effects are optional and will fail silently if files are missing.
