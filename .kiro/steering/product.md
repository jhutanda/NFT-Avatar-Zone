# Product Overview

NFT Avatar Zone is a browser-based avatar generator that merges user photos with horror-themed character templates. Users upload their photo, select from 16 spooky character templates across 8 categories (zombie, demon, ghost, witch, skull, anime-horror, cyberpunk, nft-style), and customize their avatar with real-time controls.

## Core Value Proposition

- Zero-friction creation: No signup, no installation, works instantly in browser
- Professional horror aesthetic with glassmorphism UI and atmospheric animations
- Complete customization: zoom, rotation, position, brightness, contrast, saturation
- Export and share: Download PNG, share to social media, demo NFT minting

## Target Users

- Hackathon participants and demo projects
- Social media users creating profile pictures
- NFT enthusiasts exploring avatar creation
- Creative individuals wanting spooky avatars

## Key Features

- Drag-and-drop image upload with validation
- 16 pre-designed character templates with category filtering
- Real-time preview with debounced rendering (< 100ms updates)
- Advanced editing tools: background removal, effects, filters, layers
- Responsive design for desktop, tablet, and mobile
- Optional atmospheric sound effects
- Demo NFT minting simulation
